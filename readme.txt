这个项目的几个需求：
1. 先创建一个小游戏集合页面，页面名称“益智小游戏集合”；
2. 首页先提供几个小游戏名称，并生成对应的简单图标，如：扫雷、21点、贪吃蛇、数独；并创建games子目录，稍后我们逐个完成实现。
3. 所有页面后续均需要指出触摸屏操作，适用于手机和pad，不支持键盘鼠标；
4. 后续页面需要支持离线操作，不要采用第三方链接，不需要后台服务器交互。
5. 尽量只使用html和js/css，不依赖第三方框架。
